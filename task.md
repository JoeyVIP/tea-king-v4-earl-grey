請建立一個「茶裡王V4」伯爵紅茶一頁式品牌銷售網站。使用 Next.js，部署到 Render。

## 品牌資訊
- 品牌名稱：茶裡王V4
- 主打標語：「回甘，就像現泡」
- 主要產品：伯爵紅茶（無糖、不苦澀）
- 定位：單品銷售一頁式網站

## 好茶典故
伯爵紅茶相傳因 19 世紀英國格雷伯爵的喜愛而得名。以紅茶茶葉融入愉悅的柑橘清香，成為當今世界非常受歡迎的風味紅茶。

## 情境描述
陽光和煦的午後，很適合來場茶裡王無糖伯爵紅茶搭配甜點的午茶饗宴喔。

## 網站區塊規劃
1. Hero 主視覺區：大面積茶色漸層背景，品牌名稱「茶裡王V4」+ 標語「回甘，就像現泡」，營造高質感茶飲氛圍
2. 產品特色區：三大賣點卡片（回甘口感 / 無糖健康 / 不苦澀），用簡潔圖示呈現
3. 好茶典故區：伯爵紅茶的歷史故事，搭配優雅的排版設計，營造文化底蘊
4. 午茶情境區：描繪午後茶點場景，溫暖色調，讓人產生嚮往感
5. 產品購買區：產品資訊、價格、數量選擇、加入購物車按鈕（模擬功能即可）
6. Footer：品牌資訊、聯絡方式

## 設計風格
- 配色：以茶色系為主（琥珀色、深棕、暖金），搭配乾淨的白色和淺米色背景
- 字體：優雅有質感，中文使用系統字體
- 整體氛圍：日式簡約 x 英式午茶的融合感，高質感但不花俏
- 動畫：適度的滾動淡入效果，不要過度
- 響應式：桌面和手機都要完美呈現

## 技術要求
- 使用 Next.js（App Router）
- 純前端靜態網站
- 不需要用到素材圖片，用 CSS 漸層、SVG 圖示、文字排版來營造視覺效果
- 如果需要產品圖片，使用 CSS 繪製的簡約茶瓶造型或用 placeholder
- 部署到 Render
- repo name: tea-king-v4-earl-grey
